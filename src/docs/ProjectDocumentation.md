# Grade Management System

## Project Overview

This project is a Java-based Grade Management System that stores student marks, calculates averages, assigns grades, and generates reports.

## Objectives

- Store student names and marks using arrays
- Calculate total, average, highest and lowest marks
- Generate grade categories
- Validate user input
- Provide a menu-driven interface

## Setup and Installation

1. Install Java JDK
2. Open the project in IntelliJ IDEA
3. Run GradeManagementSystem.java
4. Follow the menu options

## Code Structure

### GradeManagementSystem.java

Main program containing:

- Arrays
- Loops
- Switch statements
- Methods
- Exception handling
