# Grade Management System

## Project Overview
A Java console application that manages student marks, calculates averages, assigns grades, and generates reports.

## Features
- Add Student Marks
- Calculate Average
- Find Highest Mark
- Find Lowest Mark
- Grade Categorization
- Input Validation using try-catch
- Menu Driven System

## Technologies Used
- Java
- IntelliJ IDEA
- GitHub
